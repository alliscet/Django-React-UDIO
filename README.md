# Django-React-UDIO

Simple Django-React Application with Postgres.

Run docker-compose up followed by docker-compose run web python manage.py createsuperuser to create the admin account.
https://localhost:8000 should direct to the Django backend and https://localhost:3000 as the frontend.

https://lcoalhost:8000/peoplesearch should show a list of names and e-mails within the dabatase.
